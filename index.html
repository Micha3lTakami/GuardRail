Get Started

Installation

  1. Download the extension — Click the link from your professor to download the GuardRail extension zip. Save it to your Downloads folder.
  2. Unzip the file — Extract the zip. You should see a folder with the GuardRail extension files.
  3. Load extension in Chrome — Open Chrome → chrome://extensions. Enable "Developer mode" (top-right). Click "Load unpacked" and select the dist folder inside the GuardRail folder.
  4. Pin the extension to the toolbar — Click the puzzle-piece icon in the top-right of Chrome, find GuardRail, and click the pin icon so it appears in your toolbar for easy access.
  5. Create account — Click the GuardRail icon in your toolbar and create an account.
  6. Enter classroom code — Enter the code provided by your professor to join the pilot.
  7. Verify setup — Visit ChatGPT or Google Gemini. You should see a small GuardRail indicator in the top-right corner.

You're ready to go. GuardRail is now monitoring your AI tool usage to help you learn more effectively.

How it works

GuardRail evaluates your prompts based on learning intent:

  • Good prompt: "Can you explain how async/await works in JavaScript? I'm getting errors in my fetch request."
  • Needs improvement: "Write my JavaScript homework"

When GuardRail detects a shortcut-style prompt, it suggests ways to make it more effective for learning.

About GuardRail

GuardRail is a Chrome extension that helps you use AI tools like ChatGPT and Google Gemini more effectively. It encourages you to ask better questions that deepen your understanding instead of using AI as a shortcut.
LLMs give students information in a revolutionary way, but it can be hard to retain if not curated. GuardRail helps you learn at your own pace while giving teachers insight to adapt lessons to student needs.
Good prompts show your own thinking and ask for help with specific concepts. Examples: "Can you explain the difference between let and var in JavaScript?" or "I'm trying to center a div using flexbox but it's not working. Here's my code: [code]. What am I missing?"
